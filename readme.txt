Sistema de gestión de prestamos y devoluciones de una biblioteca universitaria. 
El sistema trabaja con una base de datos donde hay los libros tienen distintos ejemplares. 
Hay distintos tipos de usuario, por lo que dependiendo del tipo de usuario son la cantindad de ejemplares que puede solicitar para prestamo.
Si la devolución se hace vencido el plazo correspondiente, se pone una multa.
Desarrollado con PHP, HTML, CSS, JAVASCRIPT, Bootstrap.
